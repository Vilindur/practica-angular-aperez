Acerca de este Proyecto

Este proyecto fue desarrollado como una demostración de consumo de APIs en Angular. He utilizado dos APIs principales para lograr diferentes funcionalidades:


API de RAWG

La primera API que utilizamos es la API de RAWG, que proporciona información sobre videojuegos. En este proyecto, la he consumido para obtener una lista de videojuegos y filtrarlos según ciertos criterios como el nombre. Los datos se recuperan en formato JSON y se muestran de manera interactiva.

Puedes acceder a la API de RAWG y explorar más sobre los videojuegos a través del siguiente botón:

Ver documentación de RAWG

Operaciones CRUD

La segunda parte del proyecto se centra en realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) utilizando una API de artículos. Esta API gratuita permite hacer todas las operaciones CRUD de manera sencilla, y la he integrado en el proyecto para poder interactuar con artículos de prueba. Implementé las funciones para agregar, obtener, actualizar y eliminar registros a través de formularios y solicitudes HTTP en Angular.

Estas operaciones CRUD permiten realizar pruebas y ver cómo se pueden manejar datos de manera dinámica utilizando Angular.
